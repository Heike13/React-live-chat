# React-live-chat

Vite + React + TS + Socket.io
________

## What is needed 

- NPM (PNPM)
- Node.js

----

- Front install
```bash
pnpm i
```

- Back install
```bash
cd chatroom-server
pnpm i
```

- Run the server
```bash
pnpm start
```

- Run front
```bash
pnpm dev
```
