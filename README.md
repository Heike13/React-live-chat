# React-live-chat

Vite + React + TS + Socket.io


________

- To install the front packages
```bash
pnpm i
```

- To install the server : go to 
```bash
cd chatroom-server
pnpm i
```

- Run the server
```bash
pnpm start
```

- Run front
```bash
pnpm dev
```
